Detailed Description

A Wizard(magyarul: varázsló, későbbiekben is fogok rá így hivatkozni) osztály egy segédosztály, amit felhasználnak az elemek osztályai. Itt vannak azok az változók és metódusok, amiket a Brick, Ball, Paddle osztályok, valamint objektumok egyaránt felhasználnak. Úgymond egy segédosztály ami megkönnyti a többi osztály működését, és nem kell ezeketa metódusokat minden osztályba külön definiálni, megkönnyítve a program átláthatóságát és fejleszthetőségét.
